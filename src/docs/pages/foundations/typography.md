## Where each size lives

- `26` — sidebar title
- `20` — page heading
- `15` — large text field
- `14` — interface text, row label, tab
- `13` — **the control default**: button, slider, menu item
- `12` — caption, overline, hint
- `11` — helper text, field message
- `9–10` — badge

## Overline

Section titles use `.lc-overline`: 12px, uppercase, weight 700,
`letter-spacing: 0.6px`, color `--lc-muted`.

## Label opacity

Control labels in a sidebar sit at **50%** opacity (60% in an accordion header)
and return to 100% on hover. That makes the *values* — the things that change —
the highest-contrast element on screen.
